# 전처리 요약 문서

## 1. 문서 목적

본 문서는 연인 갈등 상황 대응을 위한 감정 기반 RAG 챗봇 프로젝트에서 수행한 데이터 전처리 과정과 결과를 정리하고, 각 데이터셋의 역할 및 활용 목적을 명확히 설명하기 위한 문서이다.

본 프로젝트의 전처리는 단순 데이터 정제가 아닌, **기능별(감정 분석, RAG 검색, 답변 추천)로 최적화된 데이터 구조를 설계하는 것**을 목표로 한다.

---

## 2. 전처리 목표

본 프로젝트의 핵심 기능은 다음과 같다.

1. 감정 및 갈등 위험도 분석
2. 유사 사례 검색 기반 답변 추천 (RAG)
3. 피해야 할 표현 및 대체 표현 제안

이에 따라 데이터는 기능 단위로 분리 및 재구성되었다.

---

## 3. 전처리 결과물 구성

전처리 결과물은 총 4개 파일로 구성되며, 다음 두 가지 축으로 구분된다.

### 3.1 감정 및 위험도 분석용 데이터
- `continuous_dialogue_dialogue.csv`

### 3.2 RAG 검색 및 답변 추천용 데이터

## 4. 전처리 방식 요약

### 4.1 연속 대화 데이터 전처리

기존에는 데이터 정제, 감정 라벨 정리, 구조 변환이 분리되어 있었으나, 현재는 **통합 전처리 파이프라인**으로 구성되어 일관된 데이터 생성이 가능하도록 개선하였다.

전처리 과정은 다음과 같다.

- 원본 엑셀 데이터 로드
- 대화 그룹 ID 생성
- 발화 단위 데이터 구성
- 감정 문자열 정규화
- 감정 오타 및 비정상 값 제거
- 정상 감정 필터링
- 발화 순서(turn_index) 재정렬
- 대화 단위 데이터 재구성

이를 통해 감정 분석용 데이터는 중간 산출물 없이 **최종 CSV 기준으로 바로 활용 가능하도록 설계**하였다.

---

### 4.2 공감형 대화 데이터 전처리

공감형 대화 데이터는 다음 기준으로 재구성하였다.

- 연인 관계 데이터만 필터링
- 대화 단위 문서 (RAG용) 생성
- 문맥-응답 pair 데이터 생성

또한 실제 서비스 적용 및 협업 편의성을 고려하여 **결측값을 제거하지 않고 의미 있는 기본값으로 통일**하였다.

---

## 5. 파일별 전처리 목적

### 5.1 continuous_dialogue_dialogue.csv

#### 목적
대화 단위 감정 흐름 및 전체 맥락 분석

#### 활용
- 대화 분위기 분석
- 감정 흐름 추적
- 갈등 위험도 보조 판단

---

### 5.2 continuous_dialogue_utterance.csv

#### 목적
발화 단위 감정 분석

#### 활용
- 감정 분류 모델 입력
- 위험 표현 탐지
- 사용자 감정 예측

---

### 5.3 rag_documents.csv

#### 목적
RAG 기반 유사 사례 검색을 위한 문서 데이터

#### 활용
- 상황 기반 검색
- 감정 기반 필터링
- 답변 생성 근거 제공

---

### 5.4 response_pairs.csv

#### 목적
문맥 기반 응답 추천을 위한 학습/참조 데이터

#### 활용
- 실제 응답 예시 제공
- 공감 및 대응 패턴 분석
- 답변 생성 스타일 참고

## 8. 설계 상 특징

본 전처리 설계는 다음과 같은 특징을 가진다.

- 기능별 데이터 분리 (분석 vs RAG)
- 중간 산출물 제거 → 단순한 데이터 흐름
- 서비스 적용을 고려한 결측값 처리
- 감정, 공감, 위험도 정보를 통합한 구조

---

## 9. 활용 시 주의사항

### 9.1 데이터 역할 구분
- 감정 분석: 연속 대화 데이터 사용
- RAG 및 답변 추천: 공감형 데이터 사용

### 9.2 종료 여부 컬럼 차이
- `terminated` vs `terminate` (동일 의미)

### 9.3 공감 컬럼 구분
- `listener_behavior`: 요약 수준
- `listener_empathy_tags`: 문서 단위
- `listener_empathy`: 응답 단위

---

## 10. 최종 정리

- 감정 분석 및 위험도 판단 → 연속 대화 데이터 기반
- RAG 검색 → `rag_documents.csv` 기반
- 답변 추천 → `response_pairs.csv` 기반

본 전처리 구조는 감정 분석과 RAG 기반 추천 시스템을 효율적으로 연결하기 위한 데이터 설계로 구성되었다.
